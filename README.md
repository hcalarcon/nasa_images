# 📱 React Native - App de Práctica con API Externa

Esta aplicación está pensada como proyecto educativo para que los estudiantes practiquen habilidades clave en el desarrollo de apps móviles con **React Native**. Está centrada en el consumo de una API pública y la visualización dinámica de los datos.

---

## 🎯 Objetivos pedagógicos

- Aprender el **consumo de APIs** externas usando `fetch` o `axios`.
- Entender y aplicar el manejo de la **asincronía** en JavaScript (`async/await`).
- Utilizar **renderizado condicional** para mostrar distintos contenidos según el tipo de dato.
- Navegar entre pantallas con **React Navigation**.
- Usar y configurar **bibliotecas externas** (como `react-native-vector-icons`, `translate`, etc.).
- Consultar e interpretar la **documentación oficial de APIs públicas** (ej: NASA APOD).
- Aplicar buenas prácticas de estructura, reutilización de componentes y uso de props.

---

## 🛠️ Tecnologías usadas

- [React Native](https://reactnative.dev/)
- [Expo](https://expo.dev/)
- [React Navigation](https://reactnavigation.org/)
- [API APOD de la NASA](https://api.nasa.gov/)
- Otras bibliotecas auxiliares para UI y utilidades

---

## 📸 Funcionalidades principales

- Vista del contenido del día (`ImagenDelDia`)
- Lista de días anteriores (`ImagenDeDias`)
- Vista detallada con imagen o video, explicación y acceso al recurso externo
- Mensajes condicionales según el tipo de contenido (imagen o video)
- Componente reutilizable para renderizar contenido multimedia
- Estilo responsive y modularizado
